Projet - GSB

Description du projet

GSB est un projet réalisé en deuxième année de BTS SIO. Il consiste à créer une plateforme réseau en se basant sur les recommandations de l’ANSSI.

Objectifs

  • Mettre en place une infrastructure réseau sécurisée
  • Appliquer les recommandations de l’ANSSI
  • Configurer des équipements et sécuriser les flux

Technologies utilisées

  • Firewall : Opnsense
  • Serveur : Windows Server & Linux
  • Protocoles : VPN, VLAN, Active Directory
